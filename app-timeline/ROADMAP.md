# Roadmap

This roadmap is a guide, not a contract. The project is founder-led: community input is welcome, but the maintainer decides what ships.

## Vision
A free, open, Swiss-accurate timeline tool that lets you explore cycles at multiple scales while keeping astrology “heaven” circular (wheel) and history “earth” linear (timeline).

## Now (Stabilize the foundation)
- ✅ Single ephemeris source of truth plan: Swiss Ephemeris (WASM)
- ✅ Replace legacy docked wheel + masking hacks with a popout wheel UX
- ✅ Make zoom levels coherent (generational → personal → deep cycle)

## Next (Core capabilities)
- Swiss ephemeris integration finalized:
  - consistent positions across wheel + timeline
  - caching/smoothing so scrubbing feels continuous
- Personal-scale support without forcing accounts:
  - local-first profiles (optional)
  - export/import (optional)
- Improve the historical anchor system (events under screw):
  - consistent schema
  - easy editing / import format

## Soon (Usability + learning)
- “Explain mode” overlays (teach the reading of the visualization)
- Guided presets (e.g., Saturn returns, eclipse seasons, major conjunction eras)
- Better accessibility (keyboard control, readable labels, reduced motion)

## Later (Power user tools)
- Jump-to-aspect navigator (timeline ↔ wheel)
- Multi-cycle layering (toggle sets of cycles)
- Optional plugins: curated datasets, study packs, community-curated event lanes

## Non-goals (for now)
- Accounts/login required to use the tool
- Paywalling the tool itself (teaching lives on Wix)
