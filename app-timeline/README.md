# Zodi-Yuga Timeline (Skyclock + Timeline)

A free, open, browser-based visualization tool for exploring time on a 2D timeline (earth = lines) with astrology cycles kept in the wheel (heaven = circle). It’s designed for generational-scale context **and** personal-scale exploration, without turning “heaven” into a linear list.

## What this is
- A timeline-first way to explore time cycles and historical anchors
- A popout astro-wheel that stays circular (aspects belong on the wheel)
- A “single source of truth” ephemeris engine (Swiss Ephemeris via WASM)

## What this is NOT
- A gated SaaS app
- A closed/proprietary product
- A “we sell your data” platform

## Privacy
- Default: personal inputs are processed locally in your browser.
- No accounts are required to use the tool.
- If/when we add optional exports or saved presets, we’ll keep them opt-in.

## Getting started (local dev)
Prereqs:
- Node.js (LTS recommended)

Run:
1. `npm install`
2. `npm run dev`
3. Open the localhost URL shown in the terminal
